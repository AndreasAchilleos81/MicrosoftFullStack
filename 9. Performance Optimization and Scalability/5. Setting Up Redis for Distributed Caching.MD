This guide demonstrates how to install, configure, and use Redis for distributed caching in an ASP.NET Core web application.

## üß© Step 4: Implement Controller Logic

Use `IDistributedCache` via dependency injection to interact with Redis.

### üîÑ Store Event in Cache

```csharp
[HttpPost("store")]
public async Task<IActionResult> StoreEvent([FromBody] EventModel model)
{
    var options = new DistributedCacheEntryOptions
    {
        AbsoluteExpirationRelativeToNow = TimeSpan.FromMinutes(10)
    };

    var jsonData = JsonSerializer.Serialize(model);
    await _cache.SetStringAsync(model.Id.ToString(), jsonData, options);

    return Ok($"Event {model.Id} stored in Redis.");
}
```

### üì• Retrieve Event from Cache

```csharp
[HttpGet("retrieve/{id}")]
public async Task<IActionResult> RetrieveEvent(int id)
{
    var cachedData = await _cache.GetStringAsync(id.ToString());

    if (cachedData == null)
        return NotFound("Event not found in cache.");

    var model = JsonSerializer.Deserialize<EventModel>(cachedData);
    return Ok(model);
}
```

### ‚ùå Delete Event from Cache

```csharp
[HttpDelete("remove/{id}")]
public async Task<IActionResult> RemoveEvent(int id)
{
    await _cache.RemoveAsync(id.ToString());
    return Ok($"Event {id} removed from Redis.");
}
```

### üîç Monitor Cache Status

```csharp
[HttpGet("monitor/{id}")]
public async Task<IActionResult> MonitorCache(int id)
{
    var cachedData = await _cache.GetStringAsync(id.ToString());
    return Ok(cachedData != null ? "Cache hit." : "Cache miss.");
}
```
