üîÄ Creating a Hybrid Component

Step 2: Create a HybridComponent.razor in Pages/

@if (data == null)
{
    <p>Loading data...</p>

    <ul>
        @foreach (var item in data)
        {
            <li>@item</li>
        }

@code {

üè° Injecting the Component into Home Page

Step 3: Modify Pages/Home.razor

@page "/"

<h1>Welcome to Blazor Performance App</h1>

@if (isDataLoaded)
{
    <HybridComponent />
}
else
{
    <button @onclick="LoadData">Load Data</button>
}

@code {
    private bool isDataLoaded = false;

    void LoadData()
    {
        isDataLoaded = true;
    }
}

This simulates lazy loading ‚Äî content is loaded when needed, not on initial page render.
